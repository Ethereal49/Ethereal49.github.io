# 个人主页网站

这是一个精美的个人主页网站模板，适合展示个人作品集和专业技能。

## 特点

- 响应式设计，适配各种屏幕尺寸
- 现代化 UI 界面，动画效果流畅
- 作品集展示功能，支持模态框详情查看
- 技能展示区域，直观展示专业能力
- 联系表单，方便访客与您沟通
- 加载动画和回到顶部功能

## 技术栈

- HTML5
- CSS3 (使用 CSS 变量和 Flexbox/Grid 布局)
- JavaScript (原生 JS，无框架依赖)
- Font Awesome 图标
- Lorem Picsum 占位图片服务

## 如何使用

1. 克隆仓库到本地
2. 编辑`index.html`文件，替换个人信息
3. 自定义`css/style.css`文件中的颜色和样式
4. 在`js/script.js`中添加自定义交互功能
5. 部署到您的网络服务器或 GitHub Pages

## 自定义

- 替换占位图片为您自己的照片和作品图片
- 修改个人简介、技能和作品描述
- 更新社交媒体链接
- 自定义颜色方案（在 CSS 变量中修改）
